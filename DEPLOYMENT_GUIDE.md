# 📦 Chrome Web Store Deployment Guide: Hello Prompt

Follow these steps to deploy **Hello Prompt: Invisible Intelligence** to the Chrome Web Store.

## 1. Prepare the Package
The extension is located in the `chrome-extension/` directory. You need to create a ZIP file containing all the files in that directory.

**Files to include:**
- `manifest.json`
- `background.js`
- `content.js`
- `popup.html`
- `popup.js`
- `styles.css`
- `_locales/` (and all subdirectories)
- `icons/` (and all images)

## 2. Chrome Developer Dashboard
1.  Go to the [Chrome Developer Dashboard](https://chrome.google.com/webstore/devconsole/).
2.  Sign in with your Google Account.
3.  Pay the one-time $5 developer registration fee if you haven't already.
4.  Click **+ New Item**.
5.  Upload the `hello-prompt.zip` file.

## 3. Store Listing Details

## 4. Privacy & Compliance
Based on our `COMPLIANCE_GUIDE.md`:
1.  **Single Purpose**: State that the extension provides an "Invisible Intelligence Layer to augment AI prompts."
2.  **Permission Justification**:
    *   `activeTab`: Used to interact with AI platforms only when the user explicitly triggers the enhancement.
    *   `storage`: Used to store anonymous usage stats (streaks) and user settings locally.
3.  **Data Usage**: Select **"No"** for "Collecting or using user data" (since we process in-memory and don't store user content).

## 5. Review & Publish
1.  Click **Submit for Review**.
2.  The review process typically takes 24-72 hours.
3.  Once approved, your extension will be live!
