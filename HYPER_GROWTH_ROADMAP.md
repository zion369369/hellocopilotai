# 🚀 Hyper-Growth Roadmap: Hello Prompt (FAANG-Level Scale)

This roadmap translates the "Engineering Hyper-Growth" playbook into actionable technical milestones for the **Hello Prompt** ecosystem.

## 1. Algorithmic Visibility (ASO 2.0)
*Goal: Dominate semantic search and maximize CTR.*

- [ ] **Semantic Core Expansion**: Use `generate_aso_variants.js` to target 20+ long-tail "Job-to-be-Done" keywords (e.g., "LinkedIn post formatter", "Code refactoring tool").
- [ ] **Visual CRO Audit**: Stylize screenshots with "device frames" and benefit-driven text overlays (Frame 1: The Hook, Frame 2: The Mechanism).
- [ ] **Review Gating**: Implement sentiment filtering in `content.js` to solicit 5-star reviews only after a "Value Event" (e.g., 5 successful augmentations).

## 2. Global Arbitrage (Programmatic i18n)
*Goal: Triple the TAM by localizing for non-English markets.*

- [ ] **Native i18n Architecture**: Migrate all strings to `_locales/messages.json`.
- [ ] **AI Translation Pipeline**: Update `generate_aso_variants.js` to automatically generate localized `messages.json` files for ES, PT-BR, and JA.
- [ ] **Localized Assets**: Generate localized promo tiles and screenshots for top-performing regions.

## 3. Retention Engineering (The Habit Loop)
*Goal: Transform the utility into a daily habit.*

- [ ] **The Streak Mechanic**: Implement a daily usage counter in the popup/modal. Use "Loss Aversion" messaging ("Don't break your 5-day streak!").
- [ ] **Value Realization Badges**: Award "Power User" or "Prompt Architect" badges after specific milestones to gamify the learning curve.
- [ ] **Contextual Nudges**: Send a browser notification if a user hasn't used the extension in 7 days, highlighting a new "Prompt Pack".

## 4. Viral & Referral Engines
*Goal: Achieve a K-factor > 1.0.*

- [ ] **Dual-Sided Incentives**: Implement a "Refer a Friend" system where both parties get 1 month of Pro access.
- [ ] **Artifact Loop**: Allow users to share a "Public Link" to their best augmented prompts, hosted on a conversion-optimized landing page.
- [ ] **Waiting List Scarcity**: Use a "Velvet Rope" strategy for the upcoming "Team/Enterprise" features.

## 5. Monetization & Freemium Psychology
*Goal: Maximize LTV and minimize churn.*

- [ ] **Usage Gating**: Implement a strictly enforced daily limit for free users (e.g., 10 augmentations/day) with a "Lock" UI.
- [ ] **Contextual Upsells**: Trigger the paywall at the moment of intent (e.g., when a user tries to use a "Pro" model or framework).
- [ ] **Education Bundling**: Bundle the Annual Pro Plan with a "Prompt Engineering Certification" course.
