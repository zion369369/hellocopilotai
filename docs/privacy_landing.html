Hello Copilot

Privacy Policy

Last Updated: March 24, 2026 • Version 1.0.0

Hello Copilot ("The Extension") is a browser-native AI assistant designed to provide localized, zero-friction analytical support. We prioritize user privacy and data security as foundational architectural pillars.

1. Data Collection & Usage

  • Contextual Data: The Extension processes the content of the active tab (DOM) to generate relevant AI suggestions and responses. This data is transmitted to the Gemini API via secure, encrypted channels (HTTPS).
  • No Persistence: Tab data is processed in real-time and is NOT stored on our servers. It exists only in volatile memory during the generation window.
  • Local Storage: We utilize chrome.storage.local to store your interaction preferences, daily usage counts, and episodic memories (last 10 interactions) locally on your device.

2. Permissions Justification

We believe in the principle of Least Privilege. Every permission we request is strictly necessary for our core functionality:

  • sidePanel: To provide a persistent, non-intrusive analytical interface.
  • storage: To save user preferences and local session history.
  • scripting & <all_urls>: Required to inject the 'Floating Action Button' (FAB) and extract semantic page context from diverse web environments, ensuring a "Browser-Native" experience.

3. Third-Party Services

AI generation is powered by Google's Gemini systems. By using Hello Copilot, users agree to the terms of the Gemini API environment. We do not use third-party tracking pixels, ads, or cookies.

4. Security & Compliance

  • XSS Protection: All AI-generated content is sanitized using DOMPurify before being rendered in the sidepanel.
  • Local Vault: Your episodic memory and preferences never leave your local machine.
  • No-Hassle Mode: We do not gatekeep performance or monetize user data through hidden trackers.

5. Secure Transmission & Storage

Following the **Purple Copper** policy requirements, we guarantee that all data handling meets modern cryptographic standards:

  • Encryption in Transit: All communications between the extension and the Gemini API are encrypted via HTTPS (TLS 1.3).
  • No Plaintext Exposure: Sensitive tokens and page contexts are never encoded in headers or query parameters.
  • Local-First Persistence: User-specific data (memories) is stored exclusively using `chrome.storage.local`, ensuring it remains within your browser's secure sandbox.

6. No Public Disclosure & Data Handling

In accordance with **Purple Magnesium** policies, Hello Copilot enforces strict data boundaries:

  • No Sale of Data: We never sell, trade, or rent user data to third parties.
  • No Telemetry: We do not collect web browsing activity, history, or cross-site identifiers for advertising purposes.
  • Surgical Handling: Page extraction is restricted to the active tab only and is used solely to provide user-requested AI assistance.

7. Single Purpose & Transparency

Hello Copilot complies with the **Red Magnesium** (Single Purpose) policy. Its narrow and easy-to-understand purpose is to serve as an AI-driven contextual assistant. It does not contain unrelated bundles or deceptive installation tactics.

8. User Control & Consent

You have absolute control over your data. By installing and using the extension, you consent to the processing of active tab data for AI analysis. You can revoke this consent at any time by uninstalling the extension or clearing the "Episodic Memory" from the settings menu.
